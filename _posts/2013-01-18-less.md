<h1>CSS的缺点</h1>
<h3>CSS 是一门非程序式语言</h3>
* CSS 需要书写大量看似没有逻辑的代码，不方便维护及扩展，不利于复用
* 没有变量
* 没有函数
* 没有SCOPE（作用域）	
* 非前端开发工程师来讲，往往会因为缺少 CSS 编写经验而很难写出组织良好且易于维护的 CSS 代码
